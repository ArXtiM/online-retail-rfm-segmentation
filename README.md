# RFM Customer Clustering

Краткий учебный проект по кластеризации клиентов онлайн‑магазина на основе RFM‑анализа и алгоритма K-means.  
Цель работы — сегментировать клиентов по их покупательскому поведению (давность, частота, денежная ценность покупок) и получить интерпретируемые кластеры для последующих маркетинговых решений.

## Основной функционал

- предобработка транзакционных данных (очистка пропусков, удаление аномалий и выбросов);
- расчёт RFM‑метрик для каждого клиента;
- нормализация признаков и кластеризация методом K-means;
- оценка качества кластеризации внутренними метриками (Silhouette, Davies–Bouldin, Calinski–Harabasz);
- интерпретация полученных сегментов и базовые маркетинговые рекомендации;
- веб‑интерфейс на Streamlit для интерактивной демонстрации результатов.
